# 内容发布工作流 v1.0

> 工具：MultiPost 浏览器插件
> 原则：我出内容，你一键发布

---

## 第一步：安装 MultiPost（一次性，5 分钟）

1. 打开 Chrome 网上应用店（或 Edge 加载项）
2. 搜索 "MultiPost"
3. 点击「添加至 Chrome」→ 安装完成
4. 浏览器右上角出现 MultiPost 图标 ✅

## 第二步：登录各平台（一次性，各 1 分钟）

在浏览器中逐一登录以下平台（登录一次后，插件会使用浏览器登录态）：
- [ ] 小红书（https://www.xiaohongshu.com）
- [ ] 知乎（https://www.zhihu.com）
- [ ] 抖音（https://www.douyin.com）
- [ ] 微信公众号后台（https://mp.weixin.qq.com）
- [ ] B站（选装，https://www.bilibili.com）
- [ ] LinkedIn（英文内容，选装）

> ⚠️ 登录后不要登出，保持浏览器登录态即可。MultiPost 不保存你的密码。

## 第三步：每日发布流程（日常，3 分钟）

```
我 → 每日战报中附「今日发布计划」（含内容索引）
你 → 打开 MultiPost 编辑器
你 → 按计划内容粘贴/导入
你 → 选择要发布的平台
你 → 点「一键发布」
       ↓
全平台同步发出 ✅
```
