# 产品路线图

> TrueLens - AI 内容真伪检测平台
> 制定日期：2026年7月16日

---

## 总体愿景

让任何人都能在 5 秒内知道一张图片或一段视频是真人拍摄还是 AI 生成，并获得透明、可解释的概率评分。

---

## 第一阶段：图片检测 MVP（第 1-4 周）

### 目标

上线一个可用的图片检测 Web 应用，验证市场需求。

### 功能清单

- [ ] 图片上传（拖拽 + 点击）
- [ ] AI 生成概率评分（0-100%）
- [ ] 证据展示（频域分析图、元数据状态、伪影热力图）
- [ ] 生成器指纹识别（如能识别出 Midjourney/DALL-E 等）
- [ ] 检测历史记录（本地存储）
- [ ] 深色/浅色主题

### 技术架构

```
Next.js 14 (Vercel)
    │
    ├── 前端：拖拽上传 + 结果展示
    │
    └── API Routes
            ├── DeepFlag API（主力，免费）
            ├── Hugging Face Space（ViT 模型，增强）
            └── 本地元数据分析（EXIF/C2PA）
                    │
                    └── 加权投票引擎 → 最终评分 + 证据
```

### 关键指标

- 首次加载 < 2 秒
- 检测响应 < 5 秒
- 准确率 >= 85%
- 日活用户目标：100+

### 成本预算

| 项目 | 成本 |
|------|------|
| Vercel 免费层 | ¥0 |
| DeepFlag API | ¥0 |
| Hugging Face 免费层 | ¥0 |
| 域名 | ~¥60/年 |
| **总计** | **~¥5/月** |

## 第二阶段：视频检测 + 用户系统（第 5-10 周）

### 目标

增加视频检测能力，建立用户系统，开始收集反馈。

### 功能清单

- [ ] 视频上传与检测
- [ ] 帧级伪影分析
- [ ] 用户注册/登录
- [ ] 检测历史云端同步
- [ ] 检测报告导出（PDF）
- [ ] API 接口开放（限量免费）
- [ ] Chrome 浏览器扩展（右键检测）

### 技术升级

```
新增组件：
    ├── Python FastAPI 后端（Railway）
    │       └── ACE-deepfake-detector（视频检测）
    │
    ├── Hive Moderation API（视频检测增强，$25/月）
    │
    ├── PostgreSQL（用户数据，Railway）
    │
    └── 浏览器扩展（Chrome Extension API）
```

### 关键指标

- 视频检测响应 < 30 秒
- 视频准确率 >= 85%
- 注册用户目标：500+
- API 调用量目标：1,000+/月

### 成本预算

| 项目 | 月成本 |
|------|--------|
| Vercel Pro | $20 |
| Railway（后端+数据库） | $10-15 |
| Hive Moderation Starter | $25 |
| 域名 | ~¥5/月 |
| **总计** | **~$60/月（约 ¥430）** |

## 第三阶段：商业化 + 规模化（第 11-20 周）

### 目标

实现盈利，扩展检测能力，建立品牌。

### 功能清单

- [ ] 付费订阅系统（Stripe 支付）
- [ ] 批量检测（文件夹上传）
- [ ] API 按量计费
- [ ] 实时视频流检测（直播场景）
- [ ] 音频检测（deepfake 语音）
- [ ] 检测结果可解释性增强（自然语言解释）
- [ ] 多语言支持（中/英）
- [ ] 移动端适配 / PWA

### 商业化 tiers

| 层级 | 价格 | 功能 |
|------|------|------|
| Free | ¥0 | 1次/天，图片检测，基础报告 |
| Pro | ¥39/月 | 无限检测，图片+视频，详细报告，历史导出 |
| Business | ¥199/月 | API 接口（10,000次/月），批量检测，团队协作 |
| Enterprise | 定制 | 私有部署，定制模型，SLA 保障 |

### 关键指标

- 付费转化率目标：3-5%
- 月收入目标：¥10,000+
- API 客户数：5+
- 月检测量：50,000+

### 成本预算

| 项目 | 月成本 |
|------|--------|
| 服务器升级 | $50-100 |
| 商业 API | $50-100 |
| 支付网关 | 交易额 2-3% |
| CDN | $10-20 |
| **总计** | **~$150/月（约 ¥1,080）** |
