# TrueLens 行销启动策略（v1 · 2026-07-21）

> 适用阶段：产品已完成（图片 + 视频 AI 检测、Polar 收款、反馈闭环），但**零用户、零反馈**，需要冷启动获客。
> 本期主攻人群：**个人 / 创作者 / 媒体**（不做企业风控 / 电商 / API，留待 B 端打磨后）。

---

## 1. 现状与目标

- **现状**：官网已上线（truelens.top），功能完整，但没有流量与用户。
- **核心问题**：不是产品不行，是没人知道、也没动力来用。
- **阶段目标（90 天）**：
  1. 跑通"内容飞轮"，产出首批可复用素材；
  2. 获首批 **1,000 名注册用户 / 10,000 次检测**；
  3. 沉淀 3–5 个能反复用的爆款内容模板。

---

## 2. 定位（Positioning）

- **一句话定位**：**"一秒看穿 AI 假图，给普通人也能看懂的证据。"**
- **品类**：AI 内容真伪检测 / 内容核查工具。
- **差异化**（为什么用我们而不是别家）：
  - 不只给"是 / 否"，而是给**证据链**（EXIF / C2PA 凭证 / 水印 OCR）；
  - **免费、无需注册、秒级**；
  - **中文 + 本土化**（适配国内平台分享）；
  - 结果可**一键生成分享卡片**，自带传播属性。

---

## 3. 目标用户（本期只打三类）

| 人群 | 痛点 | 我们的钩子 |
|---|---|---|
| **个人网民** | 怕被 AI 谣言图、DeepFake 视频骗（情感 / 养生 / 时政类假图） | "刷到的图拿不准？上传 2 秒识破" |
| **自媒体 / 创作者** | 发布前需核查素材是不是 AI 生成，避免"发假图翻车" | "发之前先验真，别让假图砸了号" |
| **记者 / 媒体** | 核查信源图片 / 视频真伪，做事实核查（fact-check） | "给得出证据，转发时附得上" |

**不本期打**：企业风控 / HR / 电商 / 开发者 API（留待 B 端产品打磨后单独立项）。

---

## 4. 核心卖点（已落地官网文案口径）

- 免费、无需注册、2–3 秒出结果；
- 「AI 味」评分 + **看得懂的证据**（不是黑箱）；
- 支持**图片 + 视频**，覆盖 Midjourney / Stable Diffusion / FLUX / DALL·E；
- 结果可一键生成**分享卡片**（利于自传播，也是我们的获客杠杆）。

---

## 5. 渠道地图（按"低成本、易起量、契合人群"排序）

| 优先级 | 渠道 | 内容形态 | 频率 | 目的 |
|---|---|---|---|---|
| **P0** | 小红书 | 图文笔记（"AI 假图鉴定""打假实测"） | 3–5 篇/周 | 创作者 / 女性曝光、搜索沉淀 |

| **P2** | 微博 / 粉丝群 | 互动、转发活动 | 持续 | 留存 |

## 6. 内容飞轮（可复用模板）

- **模板 A · 挑战体**：官网"测眼力"挑战区截图 + "你分得清吗？" → 引导去 truelens.top 测。
- **模板 B · 实测体**：热图 / 争议图 → 用 TrueLens 跑 → 晒证据卡片。
- **模板 C · 科普体**：5 个 AI 假图破绽（倒影 / 纹理 / 手 / 文字 / 光影）。
- **模板 D · 工具体**：盘点 AI 检测工具，TrueLens 占位（SEO 长尾）。
- **模板 E · 热点体**：蹭时事假图，做事实核查（注意合规，不碰敏感话题）。

---

## 7. 官网转化优化（本次已做）

- Hero / 特性 / 三步 / FAQ 文案改为**受众视角**（不被骗 / 核查素材 / 核查信源）；
- SEO 标题 / 描述 / 关键词扩展（含视频、深度伪造、假新闻、内容核查）；
- 移除过期"Pro 即将上线 / 沙盒测试"文案，统一为 **Pro 已上线、Polar 实收**；
- 修正免费额度描述（游客 3 次 / 注册 5 次）；
- 删除 `layout.tsx` 中伪造的 4.5 星结构化数据（避免 SEO 风险与信任问题）。

---

## 8. 衡量指标

- **北极星**：周检测次数。
- **辅助**：注册转化率（检测 → 注册）、分享率（结果 → 分享卡片）、各渠道 UTM 来源占比。
- 反馈闭环已在跑（每日 06:00 自动汇总），可据此迭代产品与文案。
