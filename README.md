# TrueLens - AI 内容真伪检测平台

> **辨识真实，守护信任**

OPC（一人公司）创业项目，由 Michael 与 AI 搭档小毕共同发起。

## 项目简介

TrueLens 是一个 AI 生成内容检测平台，帮助用户辨别图片和视频是真人拍摄还是 AI 生成，并给出概率评分和证据分析。
